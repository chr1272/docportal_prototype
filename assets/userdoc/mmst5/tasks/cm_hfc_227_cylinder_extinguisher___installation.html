HFC-227 Cylinder Extinguisher - Installation

Installation of the HFC-227 Cylinder Extinguisher.

  1. Add new teflon tape and install the Pipe Elbow (see item 2 in Figure 3) to the new bottle.
  2. Install the new HFC-227 Bottle to the bracket.
  3. Connect the Pipe Elbow to the discharge pipe with the Discharge Pipe Nut (see item 5 in Figure 1) and add new teflon tape.
  4. Install and tighten the screw for the holder (see item 3 in Figure 3).
  5. Install the Cable Connectors at the Pressure Sensor as noted down (see Figure 2).
  6. Install the black rubber cap and fix it with the screw (see Figure 2).
  7. Install the Manual Actuator and the Electrical Actuator assembly (see item 1 and 2 in Figure 1) by turning the Knurled Nut (see item 6 in Figure 1) clockwise.
  8. Switch on the Fire Suppression System.
