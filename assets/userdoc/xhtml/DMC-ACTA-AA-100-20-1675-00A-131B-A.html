Synchronization

To synchronize on demand the operational data when in docked mode, carry out the following procedure.

Responsibility Field Unit Operator
Prerequisites N/A

  1. From the menu, press the button.

    The [SYNCHRONIZATION] dialog box is displayed, indicating the status of the last synchronization.

    • A sign indicates an item successfully synchronized, meaning that this item has been downloaded and installed on the application.

    • A sign indicates an item not synchronized.

  2. Press the [UPDATE] button to launch a new synchronization.

    During downloading, a progress bar displays the amount of data currently retrieved.

    If a task order is assigned, it is downloaded so that the Field Unit Operator can acknowledge it.

    A [Synchronization] alert box is displayed when the synchronization has succeeded.

    If the current task has been closed, it will also be synchronized in docked mode (the task will be ended and removed from Mobile C2 tablet).

  3. At any time, press the [CANCEL] button or the [X] button to cancel the current item synchronization.
