Computer (Service Computer, VMD, RDU) - Installation

Installation of the Computer (Service Computer, VMD, RDU).

  1. Install the new unit by performing the removal steps in reverse order.
    Figure 1. Computer (Service Computer, VMD, RDU) system
    1
    Fixing screws
  2. Plug in the cables for a Radar Digitizing Unit System as follows:
    1. DVI port at RDU -network cable - VGA/DVI adaptor - IQ module
    2. LAN1 port at RDUx - data cable - GPUx
    3. LAN2 port at RDU - network cable - LAN port at Ethernet Switch
    4. USB1 port at RDU -data cable - IQ module
    5. Earthing connector at PDU - earthing cable - Grounding bar
    6. Pwr_01 port at RDU - power cable - Rack PDU
    7. Pwr_02 port at RDU - power cable - Rack PDU
  3. Plug in the cables for a Camera VMD Computer System follows:
    1. DVI port at RDU -network cable - VGA/DVI adaptor - IQ module
    2. LAN1 port at RDU - network cable - LAN port at Ethernet Switch
    3. USB1 port at RDU -data cable - IQ module
    4. Earthing connector at PDU - earthing cable - Grounding bar
    5. Pwr_01 port at RDU - power cable - Rack PDU
    6. Pwr_02 port at RDU - power cable - Rack PDU
  4. Plug in the cables for a Service Computer System as follows:
    1. DVI port at RDU -network cable - VGA/DVI adaptor - IQ module
    2. LAN1 port at RDU - network cable - LAN port at Ethernet Switch
    3. USB1 port at RDU -data cable - IQ module
    4. Earthing connector at PDU - earthing cable - Grounding bar
    5. Pwr_01 port at RDU - power cable - Rack PDU
    6. Pwr_02 port at RDU -power cable - Rack PDU
  • Check if LEDs are on (power, drive activity, network activity, diagnostics).
