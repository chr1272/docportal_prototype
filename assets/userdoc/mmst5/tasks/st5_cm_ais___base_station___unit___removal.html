AIS - Base Station - Unit - Removal

Removal of the AIS - Base Station - Unit.

  1. Use the Monitor and Control Tool (MCT) to shut down the FASS unit.
    Await clearance to power down the FASS unit and confirm it.
  2. Switch OFF the AIS Base Station with the power switch at the front of the device.
  3. Loosen all fixations of the AIS Base Station to the rack.
  4. Slide the rack out a few centimeters, to get access to the rear of the unit.
  5. Disconnect all cables from the AIS Base Station.
  6. Remove the AIS Base Station from the rack.
