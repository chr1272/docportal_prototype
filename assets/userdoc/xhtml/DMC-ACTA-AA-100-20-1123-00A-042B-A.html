Main Menu Bar

The main menu bar is located on the right-hand side of the screen (this location depends on the Field Unit Operator's selection through the [SWITCH] button of the [SET] menu).

It provides the Field Unit Operator with an access to the major functions of the Mobile C2 terminal application.

These functions are listed in the following table.

Table 1. Main menu functions

Button

Function

This menu allows the Field Unit Operator to inform his headquarter of his current availability.

Pressing this button gives access to the following functions:

  • Send an Operational Status Message (OP)

  • Send a Non Operational Status Message (NOP)

This menu allows the Field Unit Operator to create different types of messages.

Pressing this button gives access to the following functions:

  • Create track (CREATE)

  • Free text (TEXT)

This menu allows the Field Unit Operator to hide or display operational layers of the situation that he received from his headquarter.

This menu provides tools required to make some measures.

Pressing this button gives access to the following functions:

  • Terrain distances (DIST)

  • Terrain surface (SURF)

This menu provides access to the mailbox.

Pressing this button gives access to the following functions:

  • Sent messages (OUT)

  • Received messages (IN)

  • Deleted messages (DELETE)

  • Draft messages (DRAFT)
This menu enables the Field Unit Operator to set the parameters of his workstation and to synchronize his operational data.
Pressing this button gives access to the following functions:

  • Edit home (HOME)

  • Event log (EVENTS)

  • Brightness (BRIGHT)

  • Auto (AUTO)

  • Move vertical bar from right to left and the reverse (SWITCH)

  • Parameters (PARAMS)

  • Synchronization (SYNC)

  • Mobile C2 Logout (QUIT)
This button allows the Field Unit Operator to situate the home position on a large map when this position is outside the screen.

Pressing this button opens a large scale map window in the lower right-hand side of the screen.

A square shape in the large map represents the current screen.
