Encoder - North alignment

North alignment of the Transceiver.

  1. Set RST measurement units to correspond to the reference target data.
  2. Set “Antenna Correction” to 0 (zero) (Configuration/Interfaces/Antenna).
  3. Set “Range Adjust” to 0 (zero) (Configuration/Interfaces/Antenna).
  4. Locate the uniform, non-moving, small and stable targets with known range/bearing or GPS (>5,000 m from the radar).
  5. Use the EBL/VRM tool to show the targets on the VRM scope and measure the exact bearing. GPS positions can be converted to range and bearing from e.g. http://www.gpsvisualizer.com/calculators.
  6. Calculate and set the “Antenna Correction” (bearing correction). Antenna correction = 360° – Measured bearing + Actual bearing.
  7. Watch the PPI and inspect correct bearing to reference target, fine adjust as required.
  8. Measure the exact distance to the reference target.
  9. Calculate and set the “Range Adjust”. Range adjust = Antenna range – Actual range.
  10. Watch the PPI and inspect correct range to reference target, fine adjust as required.
