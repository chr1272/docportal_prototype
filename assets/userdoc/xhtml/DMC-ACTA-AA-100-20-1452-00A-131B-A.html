Update a Track

The Field Unit Operator updates a track by double-pressing it on the map and pressing the [UPDATE] button in the dialog box displayed.

Responsibility Field Unit Operator
Prerequisites N/A

  1. Select a track on the map.

  2. Double-press the track.

    The [TRACK INFORMATION] dialog box is displayed.

  3. Press the [UPDATE] button. The [UPDATE TRACK] dialog box is displayed.

    The field track report is already filled with the properties of the selected track (referenced by its TRACK ID).

    At any time, use the arrow buttons to change the heading (direction of movement) of a track.

  4. In the [TRACK IDENTIFICATION] Track ID field, if relevant, update the target identity.

    Change the type of target by pressing the appropriate button according to the following list:

    • SUSpect/assumed enemy - UNKnown - FRIend

    • ENemY - NEUtral - assumed FRIend

    • Unknown vehicle / Car / Truck / Military vehicle

    • Person / Person group / Boat

    • Other symbols

    • Animal / Animal herd

    • Armed / Unarmed

  5. Press the [Other Symbols] button to refine the target identification.

    The following dialog box is displayed.

    Use the scrollbar to view the symbol list, then check a box as appropriate.

    Note: The [Other Symbols] button appearance will vary according to the target identification selected in stepÂ 4, as follows:
    Table 1. Target identification
    Target identification [Other Symbols] button appearance
    UNK
    SUS
    FRI
    ENY
    NEU
    FRI?

  6. Press the [SAVE] button to save the updated field track report and close the window.

  7. Press to go to the next dialog box. The following dialog box is displayed.

  8. In the [TRACK LOCATION] heading:

    • Press the button to update the track position.

      Press [OK] to validate.

    • Press plus and minus to select the track position confidence (poor/good/very good).

      Note: Location confidence may impact on the fusion of the track with other tracks through the surveillance system.

    • Press the arrow buttons to select the direction of the track progression.

    • Press plus and minus to select the speed.

  9. Press to go to the next dialog box. The following dialog box is displayed.

  10. In the [TRACK OBSERVATIONS] heading:

    • Press the button. The [DATE TIME SELECTION] dialog box opens.

    • Update the time and press [OK] to validate.

  11. Type some comments, if needed.

  12. Press the [SEND] button to send the updated field track report and close the window.
