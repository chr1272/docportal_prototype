Receive Free Text Messages

Responsibility Field Unit Operator
Prerequisites N/A

The Field Unit Operator is notified of the message reception by a blinking letter on the left-hand side of the main window.

  1. Press the blinking letter to open the message. The [FREE TEXT] dialog box is displayed.

  2. Press:

    • The [DELETE] button to delete the current message

    • The [REPLY] button to reply to the sender of the current message

    • [X] to close the dialog box

Note: MORE THAN ONE NEW MESSAGE RECEIVED

In this case, pressing the blinking letter opens the [NEW RECEIVED MESSAGES] dialog box.

Pressing the [VIEW] button opens the message of the selected row in the [FREE TEXT] dialog box.
