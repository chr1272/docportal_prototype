Parameters Setting

To set the parameters, carry out the following procedure.

Responsibility Field Unit Operator
Prerequisites N/A

  1. From the menu, press the button.

    The [PARAMETERS] window is displayed.

    Pressing the [MANUAL] button opens the Operator Manual.

  2. From the [PARAMETERS] tab, change the following parameters as appropriate:
    • Geo Unit (DECIMAL DEG, NS WE, GEO REF, UTM M, CUSTOM, DEG MIN SEC, ...)
    • Calendar (GREGORIAN, HIJRI)
    • Speed Unit (M/S, KM/H)
    • Language (English, Arabic)
    • Time Zone (Z (UTC+0:00), A (UTC+1:00), B (UTC+2:00, ...)
    • Map (Image basemap, Raster basemap, Maritime basemap, None)

    Press the [APPLY] button to apply the new settings.

  3. Press the [ABOUT] tab to display:

    • The current release number of the application

    • The licenses of the Free and Open Source Software (FOSS) used in the application

    Press the left arrow button to go to the previous page.

    Press the [CLOSE] button to close the window.
