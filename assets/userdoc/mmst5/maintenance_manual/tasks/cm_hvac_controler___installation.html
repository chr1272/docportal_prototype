HVAC Controller - Installation

Installation of the HVAC Controller.

  1. Install the new HVAC Controller to the clip pins (see item 2 in Figure 2).
  2. Install the Display Board (see item 1 in Figure 2).
  3. Connect the cables and the connector as noted down.
  4. Close the cabinet door.
  5. Switch on power to the HVAC Controller Panel at the circuit breaker.
