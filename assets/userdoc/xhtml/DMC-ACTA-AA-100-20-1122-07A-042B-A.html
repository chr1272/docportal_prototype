Mobile Common Operational Picture (COP) Status

The Mobile COP Status indicates if the COP is totally displayed, partially displayed or not displayed.

Color Code of the Mobile COP Status

Table 1. Color code of the COP status

Color

Meaning

Area Of Operation (AOO) complete state: the entire Mobile COP of the AOO is received.

AOO limited state: only a part of the Mobile COP of the AOO is received to reduce bandwidth consumption.

The Mobile COP is unavailable since the Field Unit is out of any AOO.
Green, orange or red blinking

When a Field Unit moves from an AOO to another, the COP status keeps the color of the first AOO but blinking, until the new AOO COP is received (then it takes the new color and stops blinking).

Note: Please see Area Of Operation (AOO) and COP Status for AOO-related information.
