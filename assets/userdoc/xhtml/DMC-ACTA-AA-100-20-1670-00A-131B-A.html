Reverse Main Window Layout

This function is useful to adapt the screen for left or right handed use.

Responsibility Field Unit Operator
Prerequisites N/A

  1. From the menu, press the button.

    The following main window is displayed.

The left handed interface brings the following results:

  • Action moved to the left

  • Panic button moved to the right

  • Mail notification moved to the right
