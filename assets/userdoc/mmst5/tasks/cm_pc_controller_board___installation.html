PC Controller Board - Installation

Installation of the PC Controller Board.

  1. Mount the new module in the slot - be sure to hit the upper and lower guide for proper mounting.
  2. Keep the handles in the unlocked position until these reach the crate - press the two handles towards each other to move the module fully into the crate (until the connection is made on the motherboard).
  3. Tighten the two screws, which fix the module to the frame, with the Screwdriver PH2 x 300.
  4. Turn on power to the transceiver with the power switch on the Power Supply Unit (PSU).
    The PC starts to boot. Wait till the booting process is finished.
  5. Using the default IP address for the transceiver (this is determined by the controller board), switch “Mains On” with the Radar Service Application.
  6. Restore the backup with the Radar Service Application.
    Settings and IP addresses are now changed to their usual values.
  7. If needed, select antenna rotation, Tx and profile with the Radar Service Application.
