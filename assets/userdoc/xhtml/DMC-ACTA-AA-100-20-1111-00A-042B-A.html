Hardware Presentation

The Mobile C2 application runs on a Panasonic Toughbook CF-20 which is a tablet PC for extreme environment conditions.

The main unit can easily be detached from the keyboard base. To do this:

  1. Release the detach lock ((1) shown in Figure 2).

  2. Slide the release lever ((2) shown in Figure 2) in the direction of the arrow.

  3. Pull the main unit upwards.
