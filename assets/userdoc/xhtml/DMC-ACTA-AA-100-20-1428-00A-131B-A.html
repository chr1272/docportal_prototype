Request the End of a Task Order (adhoc task)

Responsibility Field Unit Operator
Prerequisites The task is completed.
Check

In the information bar, the color of the current task button turns from green to orange.

  1. Press the [DONE] button in the [ADHOC TASK] dialog box. A confirmation dialog box is displayed.

  2. Press the [OK] button to confirm.
