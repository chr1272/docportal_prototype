Main Function

The Mobile C2 terminal application is designed to be used by a Field Unit Operator located in a vehicle. It is deployed on ruggedized tablet laptops embedded in the vehicle.

The Mobile C2 main function is field surveillance through the Mobile C2 application running on the tablet laptops. This surveillance is achieved thanks to data exchange (through TETRA network) between the HQC2 and the Field Unit Operator, including:

  • Task orders received from the HQC2 level

  • Information about the task progress sent to the headquarter

  • Common Operational Picture (COP)

  • Blue Force Tracking

  • Field Track Reports

  • Free Text Messages

Note: The tablet laptop is connected to TETRA network in the vehicle, and to the fixed network when in docked mode.
