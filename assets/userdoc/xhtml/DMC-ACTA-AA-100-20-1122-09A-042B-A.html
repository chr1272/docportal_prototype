Field Unit Status

The Field Unit status provides information related to:

  • The current team of the Field Unit Operator

  • The current status of the Mobile C2 terminal application (Operational or Not-Operational)

Color Code of the Field Unit Status

Table 1. Color code of the Field Unit status

Color

Meaning

Field Unit operational

Field Unit in emergency mode

Field Unit not-operational

Team Information

The Field Unit Operator can display information about his current team.

To do this, press the button.

The [UNIT INFORMATION] window is displayed.

This window provides the Field Unit Operator with information related to:

  • His current status (STATUS)

  • His current team (TEAM)

  • His current user identifier (USER ID)

  • His current supervisor (TACON - Tactical Control)

  • His hierarchical supervisor (TACOM - Tactical Command)
