Received Messages

The Field Unit Operator can consult three types of received messages:
  • Orders
  • Reports
  • Free text messages

To access the full list of received messages, carry out the following procedure.

Responsibility Field Unit Operator
Prerequisites N/A

  1. From the menu, press the button. The [RECEIVED MESSAGES] window is displayed.

  2. Press:

    • the [ORDER] button to list the received messages related to the task orders received from the headquarter. The following [RECEIVED MESSAGES] dialog box is displayed.

    • the [FREE TEXT] button to list the received free text messages.

      The following [RECEIVED MESSAGES] dialog box is displayed.

    • the [ALL] button to display the full list of the received messages.

    Note: Pressing the [REPORT] button displays the following empty window.

    This is a normal behaviour, since FTRs are reports sent to the headquarter, never received.

  3. Use the navigation bar (left and right arrows in the lower part of the window) to scroll through the list of received messages:

    • To view a message, select the corresponding row, then press the [VIEW] button.

    • To delete a message:

      • Select the corresponding row, then press the [DELETE] button. A confirmation dialog box is displayed.

      • Press [OK] to confirm.
