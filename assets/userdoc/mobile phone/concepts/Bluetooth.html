Bluetooth®

Connect to any device

Bluetooth® is a wireless technology for exchanging data over short distances without physical connection

  • Turning Bluetooth® on and off
    Turn off Bluetooth when you are not using it to save battery power. Also, turned it off in places where using a wireless device is not allowed (for example on board an aircraft).
  • Sending files using Bluetooth®
    You can use Bluetooth® to send your picture, videos, and audio files to another Bluetooth® enabled device such as a phone or notebook computer.
  • Receiving files using Bluetooth®
    When a file is sent to your phone using Bluetooth® technology, a dialog box appears on your phone screen.
