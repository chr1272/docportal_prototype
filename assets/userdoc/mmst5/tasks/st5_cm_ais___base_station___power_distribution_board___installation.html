AIS - Base Station - Power Distribution Board - Installation

Installation of the AIS - Base Station - Power Distribution Board.

  1. Place the spacers from the old board on the new board.
  2. Refasten all screws, which fix the unit to the chassis (see item 3 in Figure 1).
  3. Reconnect all cables to the Power Distribution Board (see item 2 in Figure 1).
  4. Refasten the top cover of the AIS Base Station.
  5. Reinstall the AIS Base Station in the rack.
