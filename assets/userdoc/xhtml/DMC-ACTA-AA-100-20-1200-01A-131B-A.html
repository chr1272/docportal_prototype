Login

The Mobile C2 application is accessed through the following login screen.

To log in to the Mobile C2 application:

  1. Input the appropriate team identifier in the [User login] field.

    Select the current language and the current time zone.

  2. To validate, press the button to display the Mobile C2 virtual keyboard then press the [Enter] key, or press directly the button.

Pressing the button allows to shut down the tablet laptop.

Pressing the button allows to launch the Panic shutdown process.
