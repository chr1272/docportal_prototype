Remote Power Distribution 3Ph (Ground Shelter) - Configuration

Configuration of the Remote Power Distribution 3Ph (Ground Shelter).

From the KVM interface, configure the IP settings.
  1. Connect the PDU to a computer that has a terminal emulation program connected to the console port of the console server (with session settings of 9600, 8, N and 1, without flow control).
  2. Access the CLI through the console port, with Telnet or SSH or through the web manager.
  3. Type the username and password at the prompt. --> The cli-> prompt appears.
    Welcome to Avocent PM PDU host name. Type help for more information --:- / cli->
  4. At the command prompt (at the "/" level), enter wiz to view the current IP configuration.
  5. To change the IP configuration, press Tab to move through the parameters, and press Esc + Tab to edit the selected parameter.
  6. When you are finished, enter yes to confirm that all parameters are correct and to save the new parameters.
