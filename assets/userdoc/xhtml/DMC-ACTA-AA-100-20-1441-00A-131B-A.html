Create and Send Free Text Messages

Responsibility Field Unit Operator
Prerequisites N/A

  1. From the menu, press the button. The [FREE TEXT] dialog box is displayed.

  2. Press the button to select the recipients. The [RECIPIENTS SELECTION] dialog box is displayed.

    The first field (recipients field) displays the list of the selected recipients.

    The second field (textual filtering field) allows to filter the lists of recipients according to their name.

    When the textual filtering returns no recipient:

    • The list displays an empty list text.

    • The selected filter icon is greyed.

    Three tabs are available for filtering the recipients:

    • Pressing displays the list of the recipients of the teams involved in the same event.

    • Pressing displays the list of the recipients of the current region.

    • Pressing displays the list of the recipients of the neighbouring regions.

  3. Check the box(es) as appropriate to select the recipient(s), or fill the search field to filter the available recipients.

    For the current superior headquarter managing the unit (the TACON), a special icon is available in order to send the free text:

    • To all the operators of the headquarter (default mode)

    • Only to the owner of the current event

    If relevant, use the [TACON] button to select the TACON of the Field Unit in the recipient list.

  4. Press the [VALID] button to validate the recipient(s) list.

  5. In the [FREE TEXT] dialog box, check the [Acknowledgement] box for an operational acknowledge from the recipient(s).

  6. Type the message in the text area.

  7. If needed, press the [SAVE] button to save the message as a draft.

  8. Press the [SEND] button to send the message to the selected recipient(s).
