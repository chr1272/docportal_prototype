UPS Battery kit - Functional Check

Functional check of a UPS Battery Kit.

  • 2

  1. Note
    Batteries must be fully charged
    The batteries must be fully charged and the UPS must not be in battery mode to perform the battery test.
  2. Note
    Battery test
    For an accurate battery test, test the UPS with the ground shelter rack turned on.
  1. To test a UPS Battery kit:
    1. Plug the UPS Battery kit into a power outlet for 48 hours to charge the battery.
    2. Press any button to activate the menu options. Select CONTROL and then START BATTERY TEST.
      Remember: During the battery test, the front panel displays "Battery Test Pending." A failed battery test initiates a Battery Test Failed alarm.
  2. To cancel a battery test in progress:
    1. Press any button to activate the menu options.
    2. Select CONTROL, START BATTERY TEST, and then CANCEL.
