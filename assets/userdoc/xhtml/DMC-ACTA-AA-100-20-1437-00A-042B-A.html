Mobile COP Filters

The COP filters are grouped by themes:

  • [TASK] contains all the objects associated with the current task of the field unit (TASK OBJECTS).

  • [TRACKS] contains all the punctual objects continuously refreshed by the system. It filters:

    • Friendly tracks (FRIENDLY TRACKS)

    • Current hostile, presumed hostile, neutral or unknown objects (OTHER TRACKS)

    • The recent track reports sent by the Field Unit Operator to the headquarter (TRACK REPORTS)

  • [BUOYS] contains all objects, which are classified as (BUOYS).

  • [OTHER OBJECTS] contains:

    • TGOs (used to send to the field unit some events such as blocked roads) and operational zones (used to send to the field unit some zones of operational interest such as restricted areas) (TEMPORARY OBJECTS OPERATIONAL ZONES)

    • The current user AOO (AREA OF OPERATION)

  • [MAPS BASE] contains all the permanent geographical data:

    • Border Guard (BG) sites (BG SITES)

    • Critical infrastructures (CRITICAL INFRAS)

    • A complete set of maps (MAPS)

On the right-hand side of the [COP MANAGEMENT] window, check/uncheck each box as appropriate to enable/disable the corresponding filter.

When a box is checked, the [COP] button in the main menu bar turns from    to    (filtered mode).
