Introduction

'C2' is a set of applications designed to cover all needs of Command and Control (C2) activities.

The C2 set of applications includes several applications:

  • HQC2 applications, running in headquarters on workstations, and used to plan, supervise and control the Field Unit tasks.

  • Mobile C2, running on ruggedized mobile terminals in the field and executing the orders received from its headquarter.

This chapter provides an overview of the Mobile C2 application and its environment.
