AIS - Base Station - Controller - Installation

Installation of the AIS - Base Station - Controller.

  1. Fasten the mounting plate to the new Controller.
  2. Put the Controller into the chassis.
  3. Refasten the screws, which fix the Controller to the chassis (see item 2 in Figure 1).
  4. Reconnect all cables to the Controller (LAN A, LAN B, COM1, Power, Grounding).
  5. Refasten the top cover of the AIS Base Station.
  6. Reinstall the AIS Base Station in the rack.
  7. Make sure, the correct software and configuration files are loaded on the Controller (supplied by CNS).
