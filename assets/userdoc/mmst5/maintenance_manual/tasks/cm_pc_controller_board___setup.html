PC Controller Board - Setup

Setup of the Network Connection.

  1. Setup of the Transceiver Network:
    1. Turn the transceiver Main Switch on.
    2. From the main display on the transceiver go to Network Setup.
    3. Set the IP-address/subnet of LAN 2 and record in annex.
  2. Setup of the PC Network:
    1. Start the PC, which is connected by a LAN cable to the transceiver, and log on with administrative rights.
      Specific IP addresses can now be used to connect to the transceiver.
    2. In the PC: Set the date/time in the “Control Panel” to local time or to UTC (if not otherwise specified).
    3. Configure the PC LAN with TCP/IP IP address, subnet mask and gateway.
    4. Open the command window and perform a connection check by pinging the IP address of the transceiver.
