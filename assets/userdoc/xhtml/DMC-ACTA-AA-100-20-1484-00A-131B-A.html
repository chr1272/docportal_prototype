Draft Messages

The Field Unit Operator can access the list of the messages he has prepared and saved, but not yet sent. Three types of message can be listed:
  • Orders
  • Reports
  • Free text messages

To access the full list of draft messages, carry out the following procedure.

Responsibility Field Unit Operator
Prerequisites N/A

  1. From the menu, press the button.

    The [DRAFT MESSAGES] window is displayed.

  2. Press:

    • the [REPORT] button to list the draft messages related to the FTRs.

      The following [DRAFT MESSAGES] dialog box is displayed.

    • the [FREE TEXT] button to list the draft free text messages.

      The following [DRAFT MESSAGES] dialog box is displayed.

    • the [ALL] button to display the full list of the draft messages.

    For each message, the [DRAFT MESSAGES] dialog box displays:
    • The date and time (DTG TIME)
    • The message type (MSG)
    • The recipient(s) (TO)
    Note: Pressing the [ORDER] button displays the following empty window.

    This is a normal behaviour, since task orders are received from the headquarter, never sent.

  3. Use the navigation bar (left and right arrows in the lower part of the window) to scroll through the list of draft messages:
    • To delete a draft message:

      • Select the corresponding row, then press the [DELETE] button. A confirmation dialog box is displayed.

      • Press [OK] to confirm.

      To view a draft message, select the corresponding row, then press the [EDIT] button.
