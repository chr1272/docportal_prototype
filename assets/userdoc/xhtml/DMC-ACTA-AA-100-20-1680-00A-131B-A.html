Exit the Mobile C2 Application

Use the following procedure to quit the Mobile C2 application.

Responsibility Field Unit Operator
Prerequisites N/A

  1. From the menu, press the button.

    The [MOBILE-C2 LOGOUT] window is displayed.

  2. Press:

    • The icon to close the application. The tablet laptop is shutdown.

    • The icon to close the session and go back to the login screen.
