Create a Track

The Field Unit Operator creates a track using the [CREATE] button of the [FAST] menu.

The Field Unit Operator can fill a Field Track Report once he has spotted a target and must select the information about this target:
  • To define the identification of the target
  • To identify the type of target
  • To give the target geographical position
Responsibility Field Unit Operator
Prerequisites N/A

  1. From the menu, press the button. The [CREATE TRACK] dialog box is displayed.

    Note: At any time, use the arrow buttons to specify the track position on the map.

  2. In the [TRACK IDENTIFICATION] heading, define the target identity and identify the type of target by pressing the appropriate button according to the following list:

    • SUSpect/assumed enemy - UNKnown - FRIend

    • ENemY - NEUtral - assumed FRIend

    • Unknown vehicle / Car / Truck / Military vehicle

    • Person / Person group / Boat

    • Other symbols

    • Animal / Animal herd

    • Armed / Unarmed

  3. Press the [Other Symbols] button to refine the target identification.

    The following dialog box is displayed.

    Use the scrollbar to view the symbol list, then check a box as appropriate.

    Note: The [Other Symbols] button appearance will vary according to the target identification selected in step 2, as follows:
    Table 1. Target identification
    Target identification [Other Symbols] button appearance
    UNK
    SUS
    FRI
    ENY
    NEU
    FRI?

  4. Press the [SAVE] button to save the current field track report as a draft and close the window.

  5. Press to go to the next dialog box. The following dialog box is displayed.

  6. In the [TRACK LOCATION] heading:

    • Press the button to indicate the track position.

      Press [OK] to validate.

    • Press plus and minus to select the track position confidence (poor/good/very good).

      Note: Location confidence may impact on the fusion of the track with other tracks through the surveillance system.

    • Press the arrow buttons to select the direction of the track progression:

      North (0°), North-East (45°), East (90°), South-East (135°), South (180°), South-West (225°), West (270°), North-West (315°).

    • Press plus and minus to select the speed.

  7. Press the [SAVE] button to save the field track report and close the window.

  8. Press to go to the next dialog box. The following dialog box is displayed.

  9. In the [TRACK OBSERVATIONS] heading:

    • Press the button. The [DATE TIME SELECTION] dialog box opens.

    • Update the time and press [OK] to validate.

  10. Type some comments, if needed.

  11. Press the [SAVE] button to save the field track report.

  12. Press the [SEND] button to send the field track report and close the window.
