Task Type and Status

The task type and status are shown in the information bar. The task description is accessed by pressing the current task button.

The Field Unit Operator may be asked to carry out two types of tasks:

  • Standard (refers to a common task such as patrolling on a determined area). See Acknowledge and Display a Task Order (standard task).

  • Adhoc (refers to an exceptional task such as intercepting a pick-up truck or going to rescue some civilians). See Acknowledge and Display a Task Order (adhoc task).

Each one has a dedicated user interface.

Please see Table 1 for a description of the symbols identifying the different types of tasks.
