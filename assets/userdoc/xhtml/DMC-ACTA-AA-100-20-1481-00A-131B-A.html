Sent Messages

The Field Unit Operator can consult three types of sent messages:
  • Orders
  • Reports
  • Free text messages

To access the full list of sent messages, carry out the following procedure.

Responsibility Field Unit Operator
Prerequisites N/A

  1. From the menu, press the button. The [SENT MESSAGES] window is displayed.

  2. Press:

    • the [REPORT] button to list the sent messages related to the FTRs. The following [SENT MESSAGES] dialog box is displayed.

      the [FREE TEXT] button to list the sent free text messages.

      The following [SENT MESSAGES] dialog box is displayed.

      the [ALL] button to display the full list of the sent messages.

    For each message, the [SENT MESSAGES] dialog box displays:
    • The date and time (DTG TIME)
    • The message type (MSG)
    • The recipient(s) (TO)
    • The number of acknowledgments received (#)
    • The message status:
      • GREEN: the message was received by the recipients. All the message acknowledgements were received.
      • ORANGE: the message was received by at least one recipient but not all the recipients. At least one acknowledgment was received.
      • RED: the message was not received by any recipient. No acknowledgment was received.
      • GREY: no recipient acknowledgement was asked.

    Pressing the [ORDER] button displays the following empty window:

    This is a normal behaviour, since task orders are received from the headquarter, never sent.
    Use the navigation bar (left and right arrows in the lower part of the window) to scroll through the list of sent messages:

    • To view a message, select the corresponding row, then press the [VIEW] button.

    • To delete a message:

      • Select the corresponding row, then press the [DELETE] button. A confirmation dialog box is displayed.

      • Press [OK] to confirm.
