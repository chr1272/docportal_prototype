Overview

The Field Unit status provides information related to:

  • The current team of the Field Unit Operator

  • The current status of the Mobile C2 terminal application (operational, not-operational or emergency)

The status of a Field Unit is also displayed on the map, next to its symbol, as shown in the following figure (Figure 2).

The color code of the status indicator is given in the following table.

Table 1. Color code of the field unit status indicator
Status Indicator Meaning

None

The field unit is operational.

The field unit is not operational.

The field unit is in emergency mode.

The field unit is in panic mode.
