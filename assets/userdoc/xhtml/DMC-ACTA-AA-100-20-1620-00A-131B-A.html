Home Position

To set the home position, carry out the following procedure.

Responsibility Field Unit Operator
Prerequisites N/A

  1. From the menu, press the button.

    The [EDIT HOME] window is displayed.

  2. Set the home position using one of the following methods:

    • Press the button to automatically fill the home position from the current own position.

    • Press the button, then touch the screen to directly indicate the home position on the map.

      The [HOME Location] field is filled with the coordinates of the touched position.

    • Use the arrow buttons to adjust manually the home position.
