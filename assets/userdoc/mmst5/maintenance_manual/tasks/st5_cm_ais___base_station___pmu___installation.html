AIS - Base Station - PMU - Installation

Installation of the AIS - Base Station - PMU.

  1. Insert the new PMU into the DIN rail holder.
  2. Relock the DIN rail holder (close to the back plate).
  3. Reconnect all cables to the PMU (see item 2 in Figure 1).
  4. Refasten the top cover of the AIS Base Station.
  5. Reinstall the AIS Base Station in the rack.
  6. Make sure, the correct software and configuration files are loaded on the Controller (supplied by CNS).
