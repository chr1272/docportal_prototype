GPS Status and Radio Network Status

The GPS status displays the accuracy of the GPS location.

The Radio Network status displays the status of the radio link between the Mobile C2 application and the HQC2.

Color Code of the GPS Status

Table 1. Color code of the GPS status

Color

Meaning

The GPS location is very accurate: five or more satellites are visible by the GPS source with a dilution lower than 10 meters.

blinking

The GPS location is accurate: five or more satellites are visible by the GPS source with a dilution upper than or equal to 10 meters.

The accuracy of the GPS location is poor: one, two, three or four satellites are visible from the GPS source.

blinking

The accuracy of the GPS location is bad: no satellite is visible from the GPS source.

The GPS is enabled but no data is received from the GPS Source.

blinking

The accuracy of the GPS location is unknown.

The data received from the GPS Source is incorrect.

The GPS is disabled.

Color Code of the Radio Network Status

Table 2. Color code of the radio network status

Color

Meaning

The Field Unit Operator is registered on the radio network.

blinking

The Field Unit Operator received an acknowledgement of his registration on the radio network and should be registered soon.

The Field Unit Operator cannot be registered on the radio network.

  • The Field Unit Operator is waiting for registration (this status can be seen for a short time during the login/registration process).

  • The Field Unit Operator cannot be registered on the radio network.

blinking

The Field Unit Operator lost his registration to the radio network.

Or the radio network communication capabilities are degraded due to some infrastructure failures.

The Mobile C2 terminal application cannot be connected to the radio network component.

The physical link with the radio network component is removed.

blinking

The radio network component is out of order due to some hardware failures.

The radio network interface of the Mobile C2 terminal application is disabled.
