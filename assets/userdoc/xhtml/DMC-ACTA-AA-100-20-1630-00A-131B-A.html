Event Log

To access the event log, carry out the following procedure.

Responsibility Field Unit Operator
Prerequisites N/A

  1. From the menu, press the button.

    The [EVENT LOG] window is displayed.

    All types of events are displayed: ALL, ALERT, MAIL, SYSTEM, INFO.

  2. Press the [ALERT], [MAIL], [SYSTEM], and [INFO] tabs to filter the list of events according to the type.

  3. Select an item in the list, then press the [DETAILS] button for details on the selected event.

    The [EVENT DETAILS] window is displayed.
